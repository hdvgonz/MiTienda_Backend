# TIENDA API

1. Clonar Proyecto

2. Instalar los modulos de Node
```
npm install
```

3. Clonar archivo ```.env.template ``` y renombrarlo a ``` .env ```

4. Modificar variables de entorno de acuerdo a la necesidad.

5. Levantar Base de datos
```
docker-compose up -d
```

6. Levantar: 
```npm run start:dev```
